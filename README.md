# HanZex - Tech Consultancy Website

A company website for **HanZex**, a tech consultancy based in Penang, Malaysia. Built with React + Vite.

## Tech Stack

- React 19
- Vite 7
- React Router DOM

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
